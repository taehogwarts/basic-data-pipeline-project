# Basic Data Pipeline Project

## 프로젝트 개요
- 목적: 데이터 파이프라인 구축
- 수준: 기본적인 데이터 가져오기, 데이터 적재, 머신러닝 예측 모델링, 웹 API 및 대시보드 구현
